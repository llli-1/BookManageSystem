# 主要实现功能

## 主要板块
logindata.cs --记录登录用户的id和name
login.cs --登录界面

### 读者板块
reader.cs --读者主页面
reader2.cs --图书查看和借阅
reader3.cs --已借图书和归还
reader4.cs --修改读者密码

### 图书管理者板块
bookmanager.cs --图书管理者页面
bm1.cs --图书管理页面
bm11.cs --添加图书页面
bm12.cs --修改图书信息页面
bm2.cs --读者借阅信息管理页面

### 系统管理员板块
sysmanager.cs --主页面
sm1.cs --管理读者信息页面
  sm11.cs --添加读者信息页面
  sm13.cs --修改读者信息页面
sm2.cs --图书管理员管理页面
  sm21.cs --添加图书管理员页面信息
  sm22.cs --修改图书管理员信息页面
